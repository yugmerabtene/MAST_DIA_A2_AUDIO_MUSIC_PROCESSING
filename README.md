# Mastère 2 - Audio and Music Processing

Ce dépôt présente le syllabus du module **Audio and music processing** (Mastère 2), orienté vers l'analyse de signaux audio et l'application de modèles de machine learning à la musique.

## Objectifs pédagogiques

- **C3.2** Organiser un jeu de données en sous-ensembles d'entraînement et de test.
- **C3.3** Exploiter plusieurs modèles d'apprentissage supervisé pour la classification et la prédiction.
- **C3.4** Comparer les performances des algorithmes selon des métriques choisies pour sélectionner le modèle adapté.

## Durée

- **2 jours** (14 heures)

## Programme

### Jour 1

#### 1) Comprendre la structure musicale et les signaux audio (3h30)
- Concepts de théorie musicale: notes, gammes, accords, rythmes, tempo, mesures.
- Composants temporels et fréquentiels d'un signal: fréquence d'échantillonnage, amplitude, durée.
- Visualisation avec Librosa: représentations temporelles/fréquentielles et tracé de la forme d'onde.

#### 2) Extraire les caractéristiques audio (3h30)
- Extraction de features fréquentielles et harmoniques avec Librosa.
- Exemples: zero crossing rate, spectral centroid, spectral bandwidth, structure harmonique.
- Analyse de spectrogrammes et comparaison de profils sonores de différents genres.

### Jour 2

#### 1) Appliquer le machine learning à la classification de genres (3h30)
- Préparation d'un dataset de features audio.
- Entraînement de modèles de classification (k-NN, Random Forest, etc.) avec scikit-learn.
- Évaluation via des métriques classiques (dont matrice de confusion).

#### 2) Construire un moteur de recommandation musicale (3h30)
- Mesure de similarité entre morceaux via descripteurs audio.
- Création d'un mini-système de recommandation basé sur la distance des features.
- Restitution des 5 morceaux les plus proches.
