# Syllabus

## Mastère 2

### Audio and music processing

## Objectifs pédagogiques

- **C3.2** Organiser un jeu de données en sous-ensemble d'entraînement ou de test à l’aide d’un langage de programmation, de manière simple ou aléatoire en vue d’une modélisation d’apprentissage supervisée.
- **C3.3** Exploiter plusieurs modèles d’apprentissage supervisés à l’aide d’un langage de programmation permettant la classification ou la prédiction d’une variable en fonction des données disponibles.
- **C3.4** Comparer les performances des algorithmes d'apprentissages automatiques suivant les métriques choisies afin de sélectionner l’algorithme à utiliser dans le cadre de l’apprentissage supervisé.

## Durée du module

- 2 jour(s)
- Soit 14 heures

## Programme

### Jour 1

#### Comprendre la structure musicale et les signaux audio (3H30)

- Introduire les concepts de la théorie musicale : notes, gammes, accord, rythmes, tempo, mesures, etc.
- Identifier les composants temporels et fréquentiels d’un signal sonore : fréquence d’échantillonnage, amplitude, durée.
- Utiliser Librosa pour visualiser des représentations temporelles et fréquentielles d’un extrait musical.
- Tracer la forme d’onde.

#### Extraire les caractéristiques audio (3h30)

- Utiliser Librosa pour extraire des features fréquentiels et harmoniques (zero crossing rate, spectral centroid, spectral bandwidth, structure harmonique, etc.).
- Analyser les spectrogrammes.
- Comparer les profils sonores de différents genres musicaux.

### Jour 2

#### Appliquer la machine learning à la classification de genres (3h30)

- Préparer un jeu de données de features audio.
- Entraîner des modèles simples de classification musicale (k-Nearest Neighbors, Random Forest, etc.) avec scikit-learn.
- Evaluer leurs performances à l’aide de métriques classiques (matrice de confusion).

#### Construire un moteur de recommandation musicale (3h30)

- Mesurer la similarité entre morceaux via les descripteurs audio.
- Créer un mini-système de recommandation basé sur la distance des features.
- Lister les 5 morceaux les plus proches.
